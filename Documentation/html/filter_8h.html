Image histogramEqualisation (Image img)
 Perform histogram equalization on the input image. This function takes an input image and performs histogram equalization on each channel separately. It computes the histogram of each channel, then calculates the cumulative distribution function (CDF), the normalized CDF and a lookup table for mapping pixel values. Finally, it applies the lookup table to each pixel of each channel to obtain the equalized image.

Image edgeDetection (Image img, int method)
 Applies a filter to an image to detect edges using various methods. This function applies edge detection to the given input image using one of four methods: Sobel, Prewitt, Scharr, or Roberts Cross.
