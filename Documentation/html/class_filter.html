 Filter (int kernel_size)
 Filter constructor that generates a kernel using a Gaussian distribution. The kernel generated by this constructor is a square matrix of size kernel_size * kernel_size, and each element of the kernel is determined using a Gaussian distribution. The kernel is also normalized to ensure that the sum of all elements in the kernel is 1.0. This constructor initializes the kernel, kernel size, and padding size of the Filter object.
 
Image apply (Image img)
 Apply an image filter to an input image. This function takes an input image and applies a convolution filter to each pixel in each channel of the image. The filter is a 2D kernel that determines how the values of each pixel are adjusted based on the values of its neighboring pixels. The result is a new image with the filtered values.

Filter::Filter ( char *  optr)

Filter class constructor that initializes an edge detection operator. The constructor initializes the kernel matrix based on the input operator using pre-defined values for each operator. The kernel size and padding size are also set based on the kernel matrix size.

Parameters
optrA character pointer that specifies the operator for the filter kernel. Available operators are: "prewittx", "prewitty", "scharrx", "scharry", "sobelx", "sobely", "robertscrossx", "robertscrossy".

◆ Filter() [2/2]

Filter::Filter ( int  kernel_size)

Filter constructor that generates a kernel using a Gaussian distribution. The kernel generated by this constructor is a square matrix of size kernel_size * kernel_size, and each element of the kernel is determined using a Gaussian distribution. The kernel is also normalized to ensure that the sum of all elements in the kernel is 1.0. This constructor initializes the kernel, kernel size, and padding size of the Filter object.

Parameters
kernel_sizeThe size of the kernel. The kernel will be square-shaped.
